# SAYSER · Prototipos web 2026

Prototipos en evaluación para el nuevo sitio web corporativo de **SAYSER Facility Services** (Trujillo, Perú).

## Direcciones disponibles

| | Nombre | Carpeta | Característica |
|---|---|---|---|
| **A** | Corporativo cálido | `proto-1-warm/` | Banking-grade · base crema · naranja y amarillo como acentos premium |
| **C** | Cinematic de día | `proto-3-light/` | Premium luminoso · glows naranja-amarillo · 2 momentos teatrales dark |

Cada prototipo contiene 3 páginas navegables: `index.html`, `servicios.html`, `contacto.html`.

## Stack

HTML estático + Tailwind CSS (Play CDN) + GSAP + Google Fonts.
Sin build step — pueden abrirse directo en navegador o servirse vía GitHub Pages.
